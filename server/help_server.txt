
Help commands server:

       start :
              The start command active server .

       add_file :
              The "add_file" command receives the file name and the
                file path and then adds it to the database.
              example: add_file <file_name> <file path>

       remove_file :
              The remove_file command gets
                the name of the file and removes it .
              example: remove_file <file_name>

      list_files :
              The list_files command show all files with information .

      ban:
              The ban command ban get an ip and block it .
              example: ban <ip>

      unban:
              The unban command get an ip and unblock ip .
              example: unban <ip>

      exit :
              The exit command terminate server .
